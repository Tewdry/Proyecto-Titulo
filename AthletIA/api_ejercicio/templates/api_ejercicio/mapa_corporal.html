¿Cómo usar el mapa corporal AthletIA?
  • Pasa el mouse sobre un músculo para ver su nombre y una breve descripción.
  • Usa esta vista para aprender qué trabaja cada zona antes de planificar tu rutina.
  • Si quieres que AthletIA te recomiende ejercicios, vuelve a Generar Rutina y elige la opción que mejor se adapte a tu objetivo.

Consejo AthletIA: combinar grupos musculares grandes (piernas, espalda, pecho) con zonas de soporte (core, glúteos) mejora fuerza, postura y rendimiento general.
